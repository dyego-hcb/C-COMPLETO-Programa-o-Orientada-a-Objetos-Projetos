# C# COMPLETO Programação Orientada a Objetos + Projetos

Este repositório contém todo o meu trabalho e progresso no curso "C# COMPLETO Programação Orientada a Objetos + Projetos". O curso é completo e abrange desde os fundamentos da programação orientada a objetos e da linguagem C#, até tópicos mais avançados e específicos da plataforma C# e .NET.

## Seção 11: Tratamento de exceções

Nesta seção, aprendemos a forma adequada de lidar com possíveis erros em nossas aplicações. Abordamos a estrutura de exceções da linguagem C#, estrutura try-catch, bloco finally, e como criar exceções personalizadas. Vimos na prática as diferenças entre um programa adequado com lançamento de exceções comparado ao que não lança exceções.
